## Markdown 编辑器代码语法高亮支持 ##
[toc]

### 代码语法高亮 ###


书写格式为：

```
` ` ` language_key
if (condition){
return true
}
` ` `
```

在(三个反引号）之间的是代码，其中language_key的值，请参考 支持的语言和缩写标记

例如，将language_key用javascript替换，效果如下：

```javascript
if (condition){
    return true
}
```
